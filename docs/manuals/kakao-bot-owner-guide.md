# 산돌이 카카오봇 사장님 기능 설명서

산돌이 카카오톡 채널에서 식당 사장님(매니저)이 사용할 수 있는 기능의 사용 방법을 안내합니다.

- 대상 서비스: `sandol_kakao_bot_service`
- 대상 사용자: 식당 사장님 및 식당 관리 담당자

1. [로그인 방법](#1-로그인-방법)

3. [매니저 신청 방법](#3-매니저-신청-방법)

## 1. 로그인 방법

메뉴 등록 등 식당 관리 기능을 사용하려면 먼저 로그인이 필요합니다.

1. **산돌이 카카오톡 채널**에 들어갑니다.
2. 채팅방에 `로그인` 이라고 입력합니다.
3. **로그인 하러 가기** 버튼을 클릭합니다.
4. 열리는 사이트에서 **카카오로 로그인하기** 또는 **회원가입**을 클릭합니다.
   - **카카오로 로그인하기** 사용을 권장합니다.
5. **"로그인이 완료되었습니다"** 페이지가 나오면 로그인이 완료된 것입니다. 창을 닫고 카카오톡으로 돌아가 하시려던 작업을 계속하면 됩니다. (로그인이 잘 됐는지 확인하려면 채팅방에 `계정정보`를 입력하세요.)

> 로그인 세션이 만료된 경우, 채팅방에 다시 `로그인`을 입력하거나 안내 메시지의 로그인 버튼을 눌러 다시 로그인해 주세요.

---

## 2. 메뉴 등록 방법

> 메뉴 등록은 로그인 및 매니저 승인이 완료된 사용자만 이용할 수 있습니다.

1. **산돌이 카카오톡 채널**에 들어갑니다.
2. 채팅방에 `메뉴등록` 이라고 입력합니다.
3. 등록하려는 **식단 종류**(예: 중식, 석식)를 선택합니다.
4. 메뉴를 입력합니다.
   - 띄어쓰기나 쉼표로 메뉴가 구분됩니다.
   - 띄어쓰기와 쉼표를 섞어 쓸 경우, **쉼표 기준**으로 메뉴가 구분됩니다.
5. **식단 미리보기**를 통해 등록한 메뉴가 어떻게 보일지 미리 확인합니다.
6. 메뉴를 더 추가하거나 다른 종류의 식단을 추가할 것이 아니라면 **확정** 버튼을 누릅니다.

> ⚠️ **확정 버튼을 누르지 않으면 메뉴 등록이 완료되지 않습니다.** 반드시 마지막에 확정 버튼을 눌러 주세요.

---

## 3. 매니저 신청 방법

식당 관리 기능(메뉴 등록 등)을 사용하려면 해당 식당의 매니저로 등록되어야 합니다.

1. **산돌이 카카오톡 채널**에 들어갑니다.
2. 채팅방에 `매니저 신청` 이라고 입력합니다.
3. 매니저 등록을 원하는 **식당 이름**을 입력합니다. (예: 산돌식당)
4. 산돌이 운영자에게 **신청번호**를 알려줍니다.
5. 이후 운영자의 승인을 통해 식당 관리 기능을 이용할 수 있습니다.

산돌이로 직접 올려요

산돌이 카카오톡 채널에서 우리 식당의 식단을 직접 등록하고 관리하는 방법을 안내합니다. 모든 기능은 채팅방에 말을 걸듯 입력하면 시작돼요.

1 로그인 → 2 매니저 신청 → 3 메뉴 등록
STEP 1

로그인

식당 관리 기능을 사용하려면 먼저 산돌이에 내 계정을 연결해야 합니다. 최초 1회만 하면 됩니다.

  1. 산돌이 카카오톡 채널 채팅방에 들어갑니다.
  2. 채팅방에 로그인 이라고 입력합니다.
  3. 로그인 하러 가기 버튼을 누릅니다. 버튼(로그인 링크)은 10분(600초) 후 만료됩니다. 만료되면 다시 "로그인"을 입력해 주세요.
  4. 열리는 사이트에서 카카오로 로그인하기를 누릅니다. 아이디·비밀번호로 회원가입할 수도 있지만, 카카오 로그인이 가장 간편해서 추천드립니다.
  5. 카카오 동의 화면에서 전체 선택하기 → 동의하고 계속하기를 누릅니다.
  6. "로그인이 완료되었습니다" 페이지가 나오면 로그인 완료! 창을 닫고 카카오톡으로 돌아가 하시려던 작업을 계속하시면 됩니다. 로그인이 잘 됐는지 확인하고 싶다면 채팅방에 "계정정보"를 입력해 보세요.
세션이 만료됐다는 안내가 나온다면?
시간이 지나면 로그인이 풀릴 수 있습니다. 채팅방에 다시 "로그인"을 입력하고 같은 과정을 반복하면 됩니다.

STEP 2

매니저 신청

메뉴 등록 등 식당 관리 기능은 해당 식당의 매니저로 승인된 분만 사용할 수 있습니다. 로그인 후 한 번만 신청하면 됩니다.

  1. 산돌이 카카오톡 채널 채팅방에 들어갑니다.
  2. 채팅방에 매니저 신청 이라고 입력합니다.
  3. 매니저로 등록하려는 식당 이름을 입력합니다. 예: 산돌식당
  4. 접수 안내와 함께 신청 번호가 도착합니다.
  5. 산돌이 운영자에게 신청 번호를 알려주세요. 운영자가 승인하면 그때부터 식당 관리 기능을 사용할 수 있습니다.
승인은 즉시 되지 않아요.
신청 번호를 운영자에게 전달해야 승인 절차가 진행됩니다. 신청 번호를 잊었다면 접수 메시지를 다시 확인해 주세요.

STEP 3

메뉴 등록

매니저 승인이 끝났다면, 이제 채팅으로 매일의 식단을 등록할 수 있습니다.

  1. 산돌이 카카오톡 채널 채팅방에 들어갑니다.
  2. 채팅방에 메뉴등록 이라고 입력합니다.
  3. 등록하려는 식단 종류(중식/석식)를 선택합니다.
  4. 메뉴를 입력합니다. 띄어쓰기나 쉼표로 메뉴가 구분됩니다. 예: 밥, 김치, 오징어 또는 밥 김치 오징어. 둘을 섞어 쓰면 쉼표 기준으로 구분됩니다.
  5. 식단 미리보기로 등록될 모습을 확인합니다. 다른 종류(석식 등)의 메뉴를 이어서 추가할 수도 있습니다.
  6. 더 추가할 메뉴가 없다면 확정 버튼을 누릅니다.
⚠ 확정을 누르지 않으면 등록되지 않습니다!
미리보기까지 했더라도 확정 버튼을 눌러야 학생들에게 식단이 공개됩니다. 마지막에 꼭 확정을 눌러 주세요.

FAQ

자주 묻는 질문

로그인을 눌렀는데, 끝났는지 어떻게 알 수 있나요?

"로그인이 완료되었습니다" 화면이 보이면 성공입니다. 창을 닫고 카카오톡으로 돌아가시면 됩니다. 로그인이 잘 됐는지 다시 확인하고 싶다면 채팅방에 "계정정보"를 입력해 보세요.

메뉴를 입력하고 미리보기까지 봤는데 학생들에게 안 보인대요.

마지막에 확정 버튼을 눌렀는지 확인해 주세요. 확정을 누르지 않으면 임시 저장 상태로 남아 실제로 등록되지 않습니다.

"로그인 후 다시 시도해주세요"라는 메시지가 나와요.

로그인 세션이 만료된 상태입니다. 채팅방에 "로그인"을 입력해 다시 로그인한 뒤 이용해 주세요.

매니저 신청을 했는데 아직 기능을 못 써요.

운영자 승인이 필요합니다. 접수 시 받은 신청 번호를 산돌이 운영자에게 전달했는지 확인해 주세요.
